# Drug-Speak
Drug Speak is an application designed to help pharmacy students learn how to correctly pronounce drug
names. The app features a collection of drugs organized into various categories. Students can explore
these categories to view lists of drugs, with each drug entry including its name, description, molecular
formulas, and audio files provided by both female and male speakers.
The application allows students to listen to pronunciations at different speeds and record their own
attempts. It then evaluates how closely the studentʼs pronunciation matches that of the instructor, awarding
a score accordingly. For the purpose of this assignment, the evaluation functionality is simulated—students
are required to implement an algorithm that generates random scores ranging from 0 to 100.
Additionally, Drug Speak communicates with a dedicated Drug Speak Server provided by the course.
Consequently, the app must include user sign-up and sign-in features. Learning records are sent to the
backend, enabling users to view a ranking of all students using the app to study drug pronunciation.
