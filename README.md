# Drug-Speak
Drug Speak is a comprehensive mobile learning application I developed to help pharmacy students master the correct pronunciation of pharmaceutical drug names - a critical skill for professional pharmacy practice.
Key Features:
Organised Drug Library

Categorized collection of pharmaceutical drugs with detailed information
Each drug entry includes name, description, and molecular formulas
Systematic organisation for efficient learning and reference

Multi-Speaker Audio System :

High-quality pronunciation audio from both male and female instructors
Variable playback speed controls for detailed learning
Professional pharmaceutical pronunciation standards

Interactive Practice & Assessment:

Voice recording functionality for student practice attempts
Real-time pronunciation evaluation system
Scoring algorithm that compares student recordings to instructor standards (0-100 scale)

Social Learning Features:

User authentication system with secure sign-up/sign-in
Backend integration with a dedicated Drug Speak Server
Student ranking system based on pronunciation accuracy
Learning progress tracking and analytics

Technical Implementation

Frontend: Mobile application with intuitive UI/UX design
Backend Integration: RESTful API communication with the course-provided server
Audio Processing: Multi-format audio handling with speed controls
Algorithm Development: Custom scoring system for pronunciation evaluation
User Management: Secure authentication and progress tracking
